# Bê a Bá

**Como jogar**: Dado as 3 letras no topo, 
digite no campo palavras que contenham as três letras; 
Digitando uma palavra existente no dicionário português que 
contenha as 3 letras, você ganha.

## Como executar 

```bash
npm install

npm run dev
```
